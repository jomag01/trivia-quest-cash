# PWA Icons

This directory should contain app icons in the following sizes:
- icon-48x48.png
- icon-72x72.png  
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png (also used as Apple Touch Icon)
- icon-256x256.png
- icon-384x384.png
- icon-512x512.png (also maskable for Android)

## Creating Icons

1. Start with a 512x512 PNG of your app icon
2. Use a tool like https://realfavicongenerator.net/ or https://www.pwabuilder.com/imageGenerator
3. Export all sizes and place them in this folder

## Maskable Icons

For Android, icons-192x192.png and icon-512x512.png should have "safe zone" padding (the important content should be within the inner 80% of the image).

Read more: https://web.dev/maskable-icon/
